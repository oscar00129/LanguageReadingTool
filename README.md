# Description
Reading tool focused in language learning to keep track of memorized words, words currently being memorized, and words never seen until now.

# Requirements
- This app requires users.
- Users can create their own texts and work with them.
- The system will process a text provided by the user. Each word will be separated and added to a database where the user will track their progress with those words.
- The system will maintain a record of the words in the text, categorizing them as "memorized", "memorizing" or "never seen."
- Each user will have their own word register.
- Users can change the status of each word.
- Each word can be translated in app and the user can change its translation.
